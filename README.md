# risk-predictor

Risk Predictor - Health Risk Prediction System

A full-stack web application that predicts potential health risks based on user inputs using intelligent logic and provides insights to improve health awareness.

 Features
 
 User Authentication (Email & Password)
 
 Health Risk Prediction System
 
 Quiz-based health assessment
 
 Result visualization dashboard
 
 Personalized health risk output
 
 Responsive UI (React + Bootstrap)
 
 Realtime data storage (Firebase)
 
Tech Stack

Frontend:
React.js ,Bootstrap / CSS, Vite

Backend:
Django (Python)

Database:
Firebase Realtime Database

Project Structure

     project-root/
      │

      ├── frontend/   # React application

      ├── backend/    # Django project

Installation & Setup

1. Clone repository
git clone https://github.com/dishadhameliy/risk-predictor.git

cd risk-predictor
3. Backend Setup (Django)

cd backend

python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate  # Mac/Linux

pip install -r requirements.txt
python manage.py runserver

3. Frontend Setup (React)
   
cd frontend
npm install
npm run dev
Deployment

This project is deployed using Render for full-stack hosting.
